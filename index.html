The Strawberry Story

//information regarding strawberries and aim of website will be inserted here//

What my project is about: Recently, it is strawberry season again in Singapore! I want to find out where all the strawberries are and present this data in a couple of charts.

Singapore Strawberry

A bar chart would be inserted here. The title is 'Strawberry Imports to Singapore(In KG)'. It will show the top 25 countries that Singapore imported strawberries from in 2021. This will be done with the Chart class and datasets array with data from https://wits.worldbank.org/trade/comtrade/en/country/SGP/year/2021/tradeflow/Imports/partner/ALL/product/081010#

Strawberry Festival

A line chart would be inserted here. The title is 'Strawberrry production in Japan(in 1,000 metric tons). It will show the volume of strawberries produced in Japan from 2011 to 2020 and its Yokohama Strawberry festival. This will be done with the Chart class and datasets array with data from https://www-statista-com.libproxy1.nus.edu.sg/statistics/951046/japan-strawberry-production-volume/?locale=en

Link to the Yokohama Strawberry Festival website: https://www.yokohama-akarenga.jp/strawberryfes/

Global Strawberry

A world bubble map will be inserted here. The title is 'Global Strawberry Exports for the year 2022(in USD)'. It will show the trade value of the top 41 countries that export their strawberries. I will be using d3.js to build this using this tutorial by https://d3-graph-gallery.com/graph/bubblemap_template.html. The data is collected from https://wits.worldbank.org/trade/comtrade/en/country/All/year/2022/tradeflow/Exports/partner/WLD/product/081010#.
